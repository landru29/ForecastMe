ForecastMe
==========

Sport forcast

Install environment
-------------------
1. Launch npm install -g grunt grunt-cli
 
Install client
--------------
1. Go to folder client
2. Launch npm install
3. Launch bower install
4. Launch grunt

Install server
--------------
1. Go to folder server-app
2. Launch npm install
3. Go to folder server-app/seeds
4. Launch node seed.js

Launch server
-------------
1. Go to folder server-app
2. Launch npm start
